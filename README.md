# 🌐 Gestor Web para la Configuración Automatizada de Routers
## 🛡️ PROYECTO: PLAN 0 INCIDENCIAS (ASIR 2026)

### 📄 Descripción del Proyecto
[cite_start]Este sistema nace para eliminar el error humano en la administración de redes de Capa 3[cite: 17, 24]. [cite_start]A través de una interfaz web segura, permitimos realizar configuraciones masivas y precisas sin intervención manual directa en la CLI de los routers[cite: 12, 18].

### 🎯 Objetivos Principales
* [cite_start]**Automatización:** Configurar routers de forma segura mediante una plataforma centralizada[cite: 21].
* [cite_start]**Seguridad:** Acceso restringido vía VPN y credenciales específicas para garantizar la integridad del sistema[cite: 11, 26].
* [cite_start]**Trazabilidad:** Registro exhaustivo (Logs) de todos los cambios realizados, incluyendo usuario y fecha, almacenados en una base de datos[cite: 15, 25].
* [cite_start]**Eficiencia:** Reducción de incidencias mediante el uso de herramientas como Cisco NSO[cite: 12, 31].

### 📂 Organización del Repositorio
* [cite_start]**`/www`**: Interfaz de usuario y lógica de gestión web[cite: 23, 35].
* [cite_start]**`/scripts`**: Scripts de automatización y comandos de configuración para routers Cisco[cite: 13, 36].
* [cite_start]**`/db`**: Estructura de la base de datos para la persistencia de logs y usuarios[cite: 15, 37].
* [cite_start]**`/docs`**: Documentación técnica, anteproyecto y manuales de usuario[cite: 48].
